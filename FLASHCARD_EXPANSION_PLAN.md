## Implementation Guidelines

### Flashcard Structure
Each flashcard should include:
1. **Front**: Spanish word/phrase
2. **Back**: Translation + context
3. **Audio**: Native pronunciation
4. **Example**: Usage in sentence
5. **CEFR Level**: A1-C2 tag
6. **Category**: Thematic grouping

### Multi-Language Support
All flashcards support translations in:
- English (EN)
- Spanish (ES)
- Portuguese (PT)
- French (FR)
- Italian (IT)
- Nahuatl (NAH)

### Progressive Difficulty
- **A1-A2**: High-frequency everyday vocabulary
- **B1-B2**: Context-specific and abstract terms
- **C1-C2**: Academic, literary, and specialized vocabulary

### Spaced Repetition Integration
- Flashcards use SRS algorithm
- Review intervals: 1d, 3d, 7d, 14d, 30d, 90d
- XP rewards scale with level difficulty
